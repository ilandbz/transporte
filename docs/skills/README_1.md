# Skills — Sistema de Transportes Shinhua

Esta carpeta contiene los archivos de contexto (SKILL.md) para agentes IA
y desarrolladores. Leer el archivo correspondiente ANTES de generar código.

## Archivos y cuándo usarlos

| Archivo | Usar cuando... |
|---|---|
| `SKILL.md` ← **SIEMPRE primero** | Siempre. Contiene reglas globales, stack, convenciones |
| `SKILL-database.md` | Crear/modificar migraciones, modelos, seeders |
| `SKILL-backend.md` | Crear Services, Jobs, Controllers, rutas API |
| `SKILL-frontend.md` | Crear páginas Vue, componentes, layouts del Dashboard |
| `SKILL-mobile.md` | Desarrollar la app móvil Flutter/React Native |
| `SKILL-sunat.md` | Implementar emisión de CPEs con Greenter |

## Regla para agentes IA

Siempre leer en este orden:
1. `SKILL.md` (maestro)
2. El SKILL específico de la tarea

Nunca generar código sin haber leído el SKILL correspondiente.
